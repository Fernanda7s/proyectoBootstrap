# Proyecto Bootstrap

Landing page hecha con React + Vite + Bootstrap 5, basada en el boceto a mano:
Header → Hero (Body) → Carrusel → Imágenes → Barra de progreso → Tarjetas → Iconos.

## Instalación

```bash
npm install
npm run dev
```

## Estructura

Cada sección del boceto es un componente independiente dentro de `src/components`,
con su propio archivo `.jsx` y `.css`.
